## 🚀 At a Glance
Quixel Bridge Pro for Windows gives 3D artists a focused browser for the Megascans library, so scanned surfaces, 3D objects, and decals are always a search away. Instead of chasing files across folders, you preview each asset, choose a resolution, and export it into your DCC application or engine. Its standout use is with Unreal Engine 5, where bridge by quixel operates as a plugin and sends content straight into an open scene. That tight loop between browsing and placing content keeps environment work moving.

## 🎯 Quick Start
1. Open Quixel Bridge Pro and browse the Megascans library for a surface, object, or decal.
2. Pick an asset, check its preview, and set your resolution and export format.
3. For UE5, follow how to install quixel bridge in ue5: open your project, go to Plugins, enable the Bridge plugin, and restart the editor.
4. Launch Bridge from the Unreal Engine 5 toolbar to browse inside the quixel bridge unreal engine interface.
5. Download and export your asset, then drop it into the scene.

## ❓ FAQ
**Is Quixel Bridge Pro free?**
Quixel Bridge is commonly available at no cost for use with Unreal Engine. The relevant quixel bridge license terms depend on how you use the assets, so review the current terms on the official site before relying on them.

**Which versions of Windows are supported?**
It supports modern 64-bit Windows, including Windows 10 and Windows 11.

**What is Quixel Bridge?**
Quixel Bridge is a browser for the Megascans library that lets you preview photoreal assets and export them into a 3D app or game engine. In Unreal Engine 5 it works as a plugin you enable from the editor.
